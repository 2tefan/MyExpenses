Managing categories

Gestion des catégories

At the beginning, no categories are defined. You have two options for adding them, both accessible from the menu.
Au début, aucune catégorie n'est définie. vous avez deux options pour les ajouter, toutes accessibles à partir du menu.
"Import categories" allows to set up categories from files included from Grisbi in English, German or French. If with Grisbi you have already defined your own categories, you can also provide your Grisbi file, which is in XML. If you want, you can edit the file with an editor and remove any information other than the Category and Sub_category elements. Store this file as categories.xml in a folder myexpenses in the root sur une catégorie principaleof your sdcard.
"Importer les catégories" vous permet d'importer les catégories à partir des définitions standards de Grisbi en anglais, allemand ou français. Si vous avez déjà défini vos propres catégories, vous pouvez utiliser directement votre fichier, qui est d'ailleurs en format XML. Si vous le désirez, éditez votre fichier et enlevez toutes les informations autres que les blocs de catégories (Category) et de sous-catégories (Sub_category). Sauvez ce fichier sous le nom de categories.xml dans le répertoire myexpenses de la racine de votre carte SD pour pouvoir l'importer.
You now have a set up categories with subcategories you can associate with your transactions.
Vous avez maintenant un échantillon de catégories et de sous-catégories que vous pouvez utiliser pour vos opérations.
The second option is to add your own categories.  "Create new main category" brings up a dialog for providing a new category label.
La seconde option est d'ajouter vos propres catégories. "Nouvelle catégorie principale" vous ouvre une fenêtre pour entrer un nouveau libellé.
Clicking on a main category, opens up its subcategories. You select a subcategory for mapping to the current transaction by simply clicking on it. In order to map a main category, you have to call its context menu.
Cliquez sur une catégorie principale pour ouvrir ses sous-catégories. Vous pouvez selectionner une sous-catégorie pour l'opération en cours en cliquant simplement dessus. Si vous voulez utiliser la catégorie principale, il vous faudra passer par le menu contextuel.
From here you can also add new subcategories, delete a category or edit it. Subcategories have also context menus for deleting and editing.
A partir de là, vous pouvez ajouter de nouvelles sous-catégories, les effacer ou les modifier. Celles-ci ont aussi un menu contextuel afin de les effacer ou de les modifier.
